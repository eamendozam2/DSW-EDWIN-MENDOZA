Proposito:

El proposito de esta pagina es la de conocer de los frameworks CSS, las cuales disponene de una serie de clases(hojas de estilo), ya creadas que ayudan a posicionar elementos en la pagina y crear estructuras de maquetización mas o menos versatiles.

Frameworks CSS

"Es una biblioteca de estilos genericos que es usada para implementar diseños Web, algunos ejemplos son: Armazon, Baseguide, Bootstrap, BoxySheets, cascade, Chopstick". segun contenido en Wikipedia. Bibliografía: Cranford, j. (2021). Framework de CSS - Wikipedia, la enciclopedia libre. Retrieved 7 May 2021, from https://es.wikipedia.org/wiki/Framework_de_CSS

Posicionamiento WEB

"Es el conjunto de tareas y directrices que seguimos para que nuestra página web sea encontrada y posicionada en los primeros lugares de Internet cuando un usuario realiza una búsqueda a través de determinadas palabras". segun contenido wikipedia. Bibliografía: Knöbl, E. (2021). Guía de posicionamiento SEO para negocios B2B. Retrieved 7 May 2021, from https://www.titular.com/blog/guia-de-seo-para-negocios-b2b
